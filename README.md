# BobaTogether 🧋

一個現代化的飲料團購網頁應用程式，讓辦公室或朋友間的飲料訂購變得更有趣、更簡單。

## ✨ 特色

- **開團功能**：主揪可以快速建立訂單。
- **自動命名與推薦**：
  - 自動產生有趣的團購名稱（例如 "Sarah's Boba Run"）。
  - 智慧推薦功能：根據心情關鍵字（如「累」、「熱」）推薦適合的飲料。
- **完整菜單**：支援客製化冰塊與甜度。
- **響應式設計**：手機、電腦都能順暢使用。
- **無需後端**：使用 LocalStorage 暫存資料，開箱即用。

## 🚀 如何在本地端執行 (Step-by-Step)

### 1. 準備環境
確保你的電腦已安裝 [Node.js](https://nodejs.org/) (建議安裝 LTS 版本)。

### 2. 下載與安裝
將專案檔案下載到你的電腦資料夾後，打開終端機 (Terminal) 或 VS Code，進入該資料夾並執行：

```bash
npm install
```

這會下載所有需要的 React 與 Vite 工具。

### 3. 啟動網站
在終端機執行：

```bash
npm run dev
```

看到 `Local: http://localhost:5173/` 出現後，用瀏覽器打開該網址即可開始使用！

**注意：** 此版本為精簡版，所有功能皆在本地端運作，無需任何 API 金鑰。
